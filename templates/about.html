Welcome to the arXiv API

This API allows you to interact with the vast reservoir of scientific knowledge from arXiv.

About the API

The arXiv API allows you to:

  • Search for the latest publications across all research domains.
  • Search for publications on a specific topic.
  • Discover the most popular or viewed publications in a specific field.

What is arXiv?

arXiv is an open-access archive and distribution of research preprints covering the fields of mathematics, physics, astronomy, computer science, biology, finance, and statistics. arXiv is open to all and allows researchers to share their work before peer review.

API Features

Our arXiv API provides programmable access to arXiv data, enabling developers and researchers to:

  • Search for articles by keywords, publication date, author, and more.
  • Download articles in PDF format directly from the API.
  • Get recommendations based on machine learning algorithms.
  • Receive notifications for new publications in specific areas of interest.
  • Explore research trends and statistics through advanced analytics tools.

Getting Started with the API

Access our advanced search tool to start exploring the wealth of scientific data available on arXiv.

This API is built using data provided by arXiv, respecting their usage and making it more accessible for developers and researchers.
