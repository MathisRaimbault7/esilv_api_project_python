Bienvenue sur l'API arXiv

Cette API vous permet d'interagir avec le vaste réservoir de connaissances scientifiques d'arXiv.

À propos de l'API

L'API arXiv vous permet de :

  • Rechercher les dernières publications dans tous les domaines de la recherche.
  • Rechercher des publications sur un thème spécifique.
  • Découvrir les publications les plus populaires ou les plus vues dans un domaine spécifique.

Comment utiliser l'API

Utilisez les routes suivantes pour accéder aux données :

  • Dernières publications : /search/last
  • Dernières publications par thème : /search/last/[votre_theme]
  • Publications les plus populaires par thème : /search/popular/[votre_theme] (fonctionnalité à venir)

Cette API est construite en utilisant les données fournies par arXiv en respectant leur utilisation et en les rendant plus accessibles pour les développeurs et les chercheurs.
